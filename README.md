**Endless Forge**
A Minecraft bedrock addon to add new mobs, bosses, items, and structures, and biomes.

**Why I built it**
I wanted to make a Minecraft mod that will let people play for a long time because Minecraft seems too "short" after the Ender Dragon and other bosses are defeated. This mod will take more time to get better items and you can constantly improve what you have so the game feels new.

(Screenshots here)

**How to use**

To use, download as a zip file and open it in your files. Press extract all or unzip to get all the files out. Once that is done, zip the behavior_pack and resource_pack. Rename the file to a .mcaddon.

(Screenshots here)

**How to install onto Minecraft**

Open Minecraft Bedrock and double click the .mcaddon file.

(Screenshot)

Note: this project might work on Minecraft Education but some features will be broken.
